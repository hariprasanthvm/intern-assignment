App Dev Internship Assignments

This repository contains all my technical assignments for the app development internship. Since the tasks use different technologies, I have split them into separate folders so they don't interfere with each other.

Project Structure:
- 01-html-tasks: The vanilla HTML/JS drag-to-count app.
- 02-react-tasks: React component tasks (Instagram post UI, flexbox playground, etc).
- 03-react-native-tasks: Mobile app tasks (Notes app, Navigation setup, AsyncStorage).
- 04-backend-tasks: Node.js and Express backend tasks (routing, middleware, and auth).

How to run the code:
Navigate into the specific folder you want to test (for example, cd 02-react-tasks). Run 'npm install' to grab the dependencies, and then run 'npm start' to launch it locally. For the mobile apps, use 'npx expo start'.
